Funções de algumas tags

  • Table : Define a abertura de uma tabela em HTML
  • Th : Tag que define uma célula do tipo Título
  • Tr : Tag que define uma linha na tabela
  • Td : Tag que representa uma célula
  • Thead : Representa o cabeçalho da tabela
  • Tbody : Representa o corpo da tabela (Local onde a maior parte dos dados será inserida)
  • Tfoot : Representa o rodapé da tabela
  • Align : É o alinhamento dos elementos
  • Border : Representa a borda que a tabela terá
  • Caption : Tag para inserir o título da tabela
  • Rowspan : Utilizada para realizar a mesclagem de linhas
  • Colspan : Utilizada para realizar a mesclagem de colunas
  • Width : Representa a largura da tabela inserida
